- Generate a single prompt (image idea + seed phrase) each round, displayed as a prompt card.  
- Players submit a caption or a twist via a text box.  
- Upon submission the AI merges the prompt image with the caption, rendering a meme image.  
- All generated memes are shown in a gallery; the group votes (▲/▼) on each meme during a set timer.  
- After the timer, the system tallies votes, highlights the funniest meme, and shows a brief leaderboard.  
- Loop starts a new round with a fresh prompt.  
- The entire process is described in Markdown with sections **Prompt**, **Captions**, **Meme Gallery**, **Voting**, **Results**.  
- JSON payload includes `round_id`, `prompt`, `captions[]`, `memes[]`, `votes[]`, `winner`.
